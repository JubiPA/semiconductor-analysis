# Semiconductor Analysis & Machine Learning Project

This project focuses on semiconductor manufacturing data analysis using Python, machine learning, and data visualization techniques. The project was developed in Google Colab and demonstrates the complete workflow from data preprocessing and exploratory data analysis to predictive modeling and performance evaluation.

## Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Statistical analysis
- Data visualization
- Feature engineering
- Machine learning model training
- Model performance evaluation

## Technologies Used

- Python
- Google Colab
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
